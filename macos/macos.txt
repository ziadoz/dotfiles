macOS Defaults
==============

Every setting in macos.sh explained, with examples and the default value where known.


################################################################################
# General UI/UX
################################################################################

Expand save panel by default
  Key:    NSNavPanelExpandedStateForSaveMode
  Value:  true
  Default: false (panel is collapsed)
  When you save a new file (cmd+s in TextEdit, Preview, etc.) the save dialog
  normally shows a minimal box with just a filename field. Setting this expands
  it immediately to show the full file browser with sidebar, so you can navigate
  to any folder without clicking the arrow first.

Expand print panel by default
  Key:    PMPrintingExpandedStateForPrint
  Value:  true
  Default: false (panel is collapsed)
  Same idea as above but for the print dialog. Expanded view shows paper size,
  orientation, copies, and all other options immediately rather than behind a
  "Show Details" button.

Save to disk instead of iCloud by default
  Key:    NSDocumentSaveNewDocumentsToCloud
  Value:  false
  Default: true on iCloud-configured Macs
  When you create a new document in Pages, TextEdit, etc. macOS defaults to
  saving it in iCloud Drive. This changes the default location to your local
  disk. You can still manually choose iCloud when saving.

Disable the "Are you sure you want to open this application?" dialog
  Key:    LSQuarantine
  Value:  false
  Default: true
  When you download an app and open it for the first time, macOS shows a dialog
  saying "This was downloaded from the internet. Are you sure you want to open
  it?". This disables that prompt. Note: this reduces a security check. If you
  download software from untrusted sources, consider leaving this enabled.
  The unquarantine bin script gives you a per-app alternative.

Disable automatic capitalisation
  Key:    NSAutomaticCapitalizationEnabled
  Value:  false
  Default: true
  Stops macOS automatically capitalising the first letter after a full stop.
  Useful when typing code, commands, or filenames in any text field system-wide.

Disable smart dashes
  Key:    NSAutomaticDashSubstitutionEnabled
  Value:  false
  Default: true
  Stops macOS replacing -- with an en dash (–) or --- with an em dash (—).
  Useful when typing code or CLI flags (e.g. --help becoming –help breaks things
  if pasted into a terminal).

Disable automatic period substitution
  Key:    NSAutomaticPeriodSubstitutionEnabled
  Value:  false
  Default: true
  Stops macOS inserting a full stop when you double-tap the space bar.

Disable smart quotes
  Key:    NSAutomaticQuoteSubstitutionEnabled
  Value:  false
  Default: true
  Stops macOS replacing straight quotes (" ') with curly/typographic quotes (" "
  ' '). Essential for developers — pasting curly quotes into code or a terminal
  breaks things.

Disable auto-correct
  Key:    NSAutomaticSpellingCorrectionEnabled
  Value:  false
  Default: true
  Stops macOS automatically replacing words it thinks are misspelled. Useful
  if you frequently type technical terms, names, or abbreviations.


################################################################################
# Trackpad, Keyboard and Input
################################################################################

Enable tap to click
  Keys:   com.apple.driver.AppleBluetoothMultitouch.trackpad Clicking
          com.apple.mouse.tapBehavior
  Value:  true / 1
  Default: false (you must physically press the trackpad)
  Enables tap-to-click on the trackpad so a light tap registers as a click,
  for both your user account and the login screen.

Enable full keyboard access for all controls
  Key:    AppleKeyboardUIMode
  Value:  3
  Default: 2 (Tab only moves between text fields and lists)
  Lets you Tab through every interactive control in dialogs — including buttons,
  checkboxes, and radio buttons — not just text fields. Useful for confirming
  dialogs without reaching for the mouse. Press Space to activate the focused
  control.

Disable press-and-hold for keys
  Key:    ApplePressAndHoldEnabled
  Value:  false
  Default: true
  Holding a key normally shows an accent picker popup (e.g. hold 'e' to get
  é ê ë). Disabling this makes held keys repeat instead, which is more useful
  for navigation (holding arrow keys, holding backspace, etc.).

Fast keyboard repeat rate
  Keys:   KeyRepeat / InitialKeyRepeat
  Values: 1 / 10
  Default: 6 / 25 (slower)
  KeyRepeat controls how fast a held key repeats (lower is faster). InitialKeyRepeat
  controls the delay before repeating starts. 1/10 is very fast — good for
  navigating code and text quickly. If it feels too twitchy, try 2/15.


################################################################################
# Screen
################################################################################

Require password immediately after sleep
  Keys:   askForPassword / askForPasswordDelay
  Values: 1 / 0
  Default: 1 / (some grace period)
  Requires your password the moment the screen sleeps or the screensaver starts,
  with zero delay. Without this, there's often a grace period where the screen
  can be unlocked without a password.

Save screenshots to the desktop
  Key:    location
  Value:  ~/Desktop
  Default: ~/Desktop
  Sets where cmd+shift+3/4/5 screenshots are saved. Change the path if you
  prefer a dedicated Screenshots folder.

Save screenshots in PNG format
  Key:    type
  Value:  png
  Default: png
  Other options: jpg, pdf, tiff, bmp, gif. PNG is lossless and the most useful
  for screenshots. Use jpg if disk space is a concern for large screenshots.

Disable screenshot window shadow
  Key:    disable-shadow
  Value:  true
  Default: false
  When you screenshot a window (cmd+shift+4 then Space), macOS adds a drop
  shadow around it. This removes that shadow, giving you a clean window capture
  with a transparent background. Useful for documentation or sharing.


################################################################################
# Finder
################################################################################

Show path bar
  Key:    ShowPathbar
  Value:  true
  Default: false
  Shows a breadcrumb path bar at the bottom of every Finder window, showing
  the full path to the current folder (e.g. Macintosh HD > Users > you >
  Projects). You can click any segment to navigate up to that folder.

Use icon view in all Finder windows by default
  Key:    FXPreferredViewStyle
  Value:  icnv
  Default: icnv
  Sets the default view mode for new Finder windows. Options:
    icnv = icon view
    Nlsv = list view
    clmv = column view
    glyv = gallery view

Keep folders on top when sorting by name
  Key:    _FXSortFoldersFirst
  Value:  true
  Default: false
  When sorting a folder's contents by name, folders are listed first before
  files rather than mixed alphabetically with them.

Keep folders on top when sorting on the desktop
  Key:    _FXSortFoldersFirstOnDesktop
  Value:  true
  Default: false
  Same as above but applies to icons on the desktop.

Search the current folder by default
  Key:    FXDefaultSearchScope
  Value:  SCcf
  Default: SCev (search this Mac)
  When you search in Finder (cmd+f), this sets the default scope to the folder
  you're currently in rather than the whole Mac. You can still change it per
  search using the scope bar.

Disable the warning when changing a file extension
  Key:    FXEnableExtensionChangeWarning
  Value:  false
  Default: true
  Removes the confirmation dialog ("Are you sure you want to change the extension
  from .txt to .md?") when you rename a file and change its extension.

Open new Finder windows to the home folder
  Keys:   NewWindowTarget / NewWindowTargetPath
  Values: PfHm / file://${HOME}/
  Default: PfRe (Recents)
  Sets the default location for new Finder windows. PfHm is your home folder.
  Other options:
    PfDe = Desktop
    PfDo = Documents
    PfLo = custom path (set via NewWindowTargetPath)

Show external drives and removable media on the desktop
  Keys:   ShowExternalHardDrivesOnDesktop / ShowRemovableMediaOnDesktop
  Values: true / true
  Default: true / true
  External hard drives (USB, Thunderbolt) and removable media (SD cards, USB
  sticks) will appear as icons on the desktop when connected. Internal drives
  and network servers are kept off the desktop.

Keep internal drives and servers off the desktop
  Keys:   ShowHardDrivesOnDesktop / ShowMountedServersOnDesktop
  Values: false / false
  Default: false / false
  Internal volumes (Macintosh HD) and mounted network servers won't clutter
  the desktop. They're still accessible via Finder sidebar.

Avoid creating .DS_Store files on network or USB volumes
  Keys:   DSDontWriteNetworkStores / DSDontWriteUSBStores
  Values: true / true
  Default: false / false
  .DS_Store files store folder view settings (icon positions, view mode,
  background colour). On network shares they're visible to others and can cause
  noise in version control. On USB drives they litter the drive for other
  operating systems. Disabling them means custom icon arrangements on those
  volumes won't be remembered, but that's rarely a problem.

Enable spring loading for directories with reduced delay
  Keys:   com.apple.springing.enabled / com.apple.springing.delay
  Values: true / 0.3
  Default: true / 0.5
  Spring loading lets you drag a file over a folder and have it open
  automatically after the delay, so you can drop into a subfolder without
  navigating there first. The default delay is 0.5 seconds. 0.3 is snappier
  without being too reactive. Setting it to 0 makes it instant (usually
  too aggressive).


################################################################################
# Dock
################################################################################

Use scale effect when minimising windows
  Key:    mineffect
  Value:  scale
  Default: genie
  Controls the animation when minimising a window to the Dock. Scale shrinks
  the window straight down. Genie swoops it into the Dock with a wave effect.
  Scale is faster and less distracting.

Minimise windows into their application's icon
  Key:    minimize-to-application
  Value:  true
  Default: false
  When you minimise a window it disappears into the app's Dock icon rather than
  creating a separate thumbnail in the Dock. Keeps the Dock tidier. To restore,
  right-click the app icon and choose the window.

Show indicator lights for open applications
  Key:    show-process-indicators
  Value:  true
  Default: true
  Shows a small dot below each app icon in the Dock when that app is running.

Speed up Mission Control animations
  Key:    expose-animation-duration
  Value:  0.1
  Default: ~0.25
  Reduces the animation time when opening Mission Control (ctrl+up or three-
  finger swipe up). 0.1 is near-instant. Set higher if you want a smoother
  transition.

Don't rearrange Spaces based on recent use
  Key:    mru-spaces
  Value:  false
  Default: true
  By default macOS reorders your Spaces so the most recently used one moves
  to the front. This disables that so your Spaces stay in a fixed order, which
  makes muscle memory for Space positions reliable.

Remove the auto-hiding Dock delay
  Key:    autohide-delay
  Value:  0
  Default: ~0.5
  When the Dock is set to auto-hide, there's normally a delay before it appears
  when you move to the screen edge. Setting this to 0 makes it appear instantly.

Remove the Dock show/hide animation
  Key:    autohide-time-modifier
  Value:  0
  Default: ~0.5
  Controls how long the Dock slide-in/out animation takes. 0 makes it
  appear and disappear instantly.

Automatically hide and show the Dock
  Key:    autohide
  Value:  true
  Default: false
  Hides the Dock until you move your cursor to the bottom of the screen. Gives
  you more vertical screen space.

Make hidden application icons translucent
  Key:    showhidden
  Value:  true
  Default: false
  When you hide an application (cmd+h), its Dock icon becomes semi-transparent.
  Gives you a visual indicator of which apps are hidden vs. visible.

Don't show recent applications in the Dock
  Key:    show-recents
  Value:  false
  Default: true
  Removes the "Recent Applications" section at the right end of the Dock that
  shows apps you've opened recently but haven't pinned to the Dock.


################################################################################
# Terminal
################################################################################

Only use UTF-8 in Terminal.app
  Key:    StringEncodings
  Value:  4 (UTF-8)
  Default: may vary
  Forces Terminal.app to use UTF-8 encoding, which ensures characters like
  accented letters, emoji, and symbols display correctly.

Enable Secure Keyboard Entry in Terminal.app
  Key:    SecureKeyboardEntry
  Value:  true
  Default: false
  Prevents other applications from reading what you type in Terminal. Useful
  if you type passwords or secrets in the terminal. Can interfere with some
  clipboard or automation tools.

Disable line marks in Terminal.app
  Key:    ShowLineMarks
  Value:  0
  Default: 1
  Removes the small triangular marks in the Terminal scrollbar that indicate
  where each command prompt was. Cleans up the scrollbar appearance.


################################################################################
# Time Machine
################################################################################

Don't prompt to use new drives as backup volumes
  Key:    DoNotOfferNewDisksForBackup
  Value:  true
  Default: false
  Stops macOS showing a dialog asking if you want to use every newly connected
  external drive as a Time Machine backup volume. Useful if you connect drives
  for other purposes frequently.


################################################################################
# Activity Monitor
################################################################################

Show the main window when launching Activity Monitor
  Key:    OpenMainWindow
  Value:  true
  Default: false (only the Dock icon shows)
  Opens the process list window automatically when you launch Activity Monitor,
  rather than just showing the icon in the Dock.

Show all processes
  Key:    ShowCategory
  Value:  0
  Default: 1 (My Processes)
  Shows all running processes in Activity Monitor rather than filtering to only
  processes owned by your user. Lets you see system processes, daemons, etc.

Sort by CPU usage
  Keys:   SortColumn / SortDirection
  Values: CPUUsage / 0 (descending)
  Default: CPUUsage / 0
  Sorts the process list by CPU usage descending so the most CPU-hungry processes
  are at the top when you open Activity Monitor.


################################################################################
# TextEdit
################################################################################

Use plain text mode for new documents
  Key:    RichText
  Value:  0
  Default: 1 (rich text / .rtf)
  Makes TextEdit open and create documents in plain text (.txt) mode by default
  rather than rich text (.rtf). Useful for quick notes and editing config files.

Open and save files as UTF-8
  Keys:   PlainTextEncoding / PlainTextEncodingForWrite
  Values: 4 / 4
  Default: may vary
  Forces TextEdit to read and write plain text files as UTF-8, which is the
  correct encoding for most modern text files.


################################################################################
# Software Update
################################################################################

Enable automatic update checks
  Key:    AutomaticCheckEnabled
  Value:  true
  Default: true
  Ensures macOS checks for system software updates automatically.

Check for updates daily
  Key:    ScheduleFrequency
  Value:  1
  Default: 7 (weekly)
  Checks for software updates every day rather than once a week, so you hear
  about updates sooner.

Download updates in the background
  Key:    AutomaticDownload
  Value:  1
  Default: varies
  Downloads available updates automatically in the background so they're ready
  to install when you choose to, without waiting for the download.

Install system data files and security updates
  Key:    CriticalUpdateInstall
  Value:  1
  Default: 1
  Automatically installs security updates and system data files (like malware
  definitions) without prompting. Does not auto-install major macOS updates.


################################################################################
# Photos
################################################################################

Prevent Photos from opening when devices are plugged in
  Key:    com.apple.ImageCapture disableHotPlug
  Value:  true
  Default: false
  Stops Photos.app from launching automatically whenever you plug in an iPhone,
  iPad, camera, or SD card. You can still open Photos manually to import.


################################################################################
# Google Chrome
################################################################################

Disable over-sensitive backswipe
  Keys:   AppleEnableSwipeNavigateWithScrolls
          AppleEnableMouseSwipeNavigateWithScrolls
  Values: false / false
  Default: true
  Chrome's swipe-to-go-back/forward gesture on trackpads and Magic Mouse is
  very sensitive and triggers accidentally during normal scrolling. Disabling
  it stops the page from navigating when you didn't intend it to.

Use the system print dialog
  Key:    DisablePrintPreview
  Value:  true
  Default: false
  Uses macOS's native print dialog instead of Chrome's built-in one. The native
  dialog gives you access to all printer options, PDF saving, and is consistent
  with other apps.

Expand print dialog by default
  Key:    PMPrintingExpandedStateForPrint2
  Value:  true
  Default: false
  Shows the full print options immediately when you print from Chrome rather
  than showing a simplified view first.


################################################################################
# Transmission
################################################################################

Use ~/Torrents for completed downloads
  Keys:   DownloadFolder / DownloadLocationConstant
  Values: ~/Torrents / true
  Default: false (asks each time or uses a custom path)
  Creates ~/Torrents if it doesn't exist and sets it as the fixed download
  destination for completed torrents.

Don't prompt before downloading
  Keys:   DownloadAsk / MagnetOpenAsk
  Values: false / false
  Default: true / true
  Starts downloading immediately when you open a torrent file or magnet link
  rather than asking you to confirm the download location each time.

Hide the donate message and legal disclaimer
  Keys:   WarningDonate / WarningLegal
  Values: false / false
  Default: true / true
  Suppresses the recurring donate prompt and the legal disclaimer that Transmission
  shows on first launch.
